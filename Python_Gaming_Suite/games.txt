Title: Guess the Number Game

Overview:
Guess the Number is a simple yet engaging game where players attempt to guess a randomly generated number within a specified range. This Python-based game is implemented using the Pygame library, offering a graphical user interface for an interactive gaming experience.

Game Objective:
The objective of the game is straightforward: guess the hidden number correctly within the fewest attempts possible.

Game Features:

Random Number Generation: The game generates a random number between 1 and 100 at the beginning of each session. This is the number that the player needs to guess.
User Input: Players can input their guesses using the keyboard. The "Enter" key submits the guess.
Visual Feedback: The game provides visual feedback on each guess, indicating whether the player should try a higher or lower number.
Attempt Counter: An attempt counter keeps track of the number of guesses the player has made.
Game Over: The game ends when the player correctly guesses the number or decides to quit. A Game Over message displays the result.
Play Again: After the game ends, players have the option to play again or quit the application.
How to Play:

Launch the game.
A random number between 1 and 100 is generated and kept secret.
Type your guess using the keyboard and press "Enter" to submit your guess.
The game provides feedback, telling you to try a higher or lower number based on your guess.
Keep guessing until you correctly guess the hidden number or decide to quit.
After the game ends, you can choose to play again or exit the application.
